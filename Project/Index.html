How To Play

The game hides three gifts (at random each time the game is loaded).

Your task is to find the gifts.

The user clicks inside any square for a guess. The square flips to show what’s hidden inside. In response to your guess, you will either see the “Gift” or “Miss” in the corresponding square. When you find all the gifts, the game ends by displaying your rating. In case you find the danger icon (only one such icon is hidden in the grid, at random) the game will reset and all progress is lost.

The game will have three levels: 3x3 grid, 4x4 grid and 6x6 grid.
